# CinePulse - مستكشف الأفلام الذكي 🎬

CinePulse هو تطبيق ويب عصري يسمح للمستخدمين باستكشاف الأفلام والمسلسلات بواجهة مستخدم غامرة وتصميم فائق الجودة. تم تصميم التطبيق مع التركيز على الجماليات البصرية وسهولة الاستخدام.

## 🚀 المميزات الرئيسية
- **واجهة مستخدم غامرة (Premium UI):** استخدام تأثيرات Glassmorphism وتدرجات لونية نابضة بالحياة.
- **عرض الأفلام بشكل تفاعلي:** بطاقات أفلام تتفاعل مع حركة المؤشر مع تأثيرات تحويم (Hover Effects) سلسة.
- **البحث اللحظي (Live Search):** محرك بحث سريع يقوم بتصفية الأفلام فور الكتابة.
- **نظام التصفية (Filtering System):** تصفية الأفلام حسب النوع (أكشن، دراما، خيال علمي).
- **تصميم متجاوب (Responsive Design):** يعمل بشكل مثالي على الهواتف والأجهزة اللوحية والحواسيب.
- **رسوم متحركة متطورة:** استخدام CSS Animations ومراقب التقاطع (Intersection Observer) لتجربة حيوية.

## 🛠 التكنولوجيا المستخدمة
- **HTML5:** الهيكل البنائي مع مراعاة معايير SEO.
- **CSS3:** استخدام المتغيرات (CSS Variables)، Flexbox، Grid، و Keyframes.
- **JavaScript (Vanilla):** إدارة البيانات، البحث، والتصفية دون الحاجة لمكتبات خارجية.
- **Google Fonts:** خطوط Outfit و Tajawal للتميز الطباعي.
- **Font Awesome:** للأيقونات التفاعلية.

## 📝 كيفية الاستخدام
1. قم بفتح ملف `index.html` في متصفحك.
2. استخدم شريط البحث في الأعلى للبحث عن أفلام محددة.
3. استخدم زر التصنيفات لتصفية الأفلام حسب نوعها.
